# ENGO 551 Lab 5 – IoT GeoWeb App

Lab 5 Submission for Charu. 


This app connects to an MQTT broker to publish and receive messages. Users can send text messages or share their location as GeoJSON, which is displayed on a map with random temperature visualizations.  

Files:
- index.html – main web application  
- static/styles.css – styling for the app  

Acknowledgments: MQTT WebSocket functionality adapted from  
http://www.steves-internet-guide.com/download/javascript-websockets/
